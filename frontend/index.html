How the distribution works

  • FastAPI does not talk to MongoDB directly from each endpoint.
  • A routing layer decides the correct shard using the product ID.
  • Even product IDs and odd product IDs end up in different databases.
  • Logs show which shard each product request uses.
  • A 3-node MongoDB replica set can be used underneath to demonstrate replication and failover.
